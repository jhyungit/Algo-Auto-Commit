### 풀이 접근

**핵심 아이디어:** 모든 신호등 주기의 **최소공배수(LCM)** 까지만 검사하면 충분하다.

각 신호등은 자기 주기 `Pᵢ = Gᵢ + Yᵢ + Rᵢ` 로 독립적으로 반복된다. 시각 `t` 에서의 상태 조합은 `(t % P₁, t % P₂, ..., t % Pₙ)` 이고, `L = LCM(P₁, ..., Pₙ)` 마다 모든 상태 조합이 정확히 반복된다.

→ **시각 1 ~ L 안에 "전부 노랑"이 없으면 영원히 없다 → -1 반환**

### 구현 포인트

1. 각 신호등의 한 주기 패턴을 문자열로 생성: `'G'*g + 'Y'*y + 'R'*r`
2. 모든 주기의 LCM을 `reduce(lcm, periods)` 로 계산
3. 시각 1부터 LCM까지 순회하며, `(time - 1) % len(light)` 로 인덱싱
4. `all()` 의 단락 평가로 빠르게 거름

### 학습 노트

- **`gcd × lcm = a × b`** 공식으로 두 수의 lcm 계산: `a * b // gcd(a, b)`
- **`reduce`** 로 여러 수의 lcm을 누적 계산 (lcm은 결합법칙 성립)
- **mod 연산**으로 한 주기 패턴을 무한 반복 표현 → 메모리 절약
- Python 3.8 환경에서는 `math.lcm` 이 없어 `gcd` + `reduce` 조합 필요

### 이전 풀이와 비교

기존 풀이는 신호등 패턴을 `temp *= 1_000_000` 으로 물리적으로 복제해 메모리가 컸다 (645 MB). 새 풀이는 한 주기만 저장하고 LCM을 상한으로 사용해 메모리를 **9.24 MB** 까지 줄였다 (약 70배 감소).
